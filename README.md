# tonyshen2003.github.io
##树莓派·苏州中学树莓社章程（草案）
2018年9月15日苏州中学树莓社第一届社员大会会议通过；自2018年9月20日苏州中学树莓社参与纳新宣传开始正式施行。
经2019年8月15日第一届树莓派组织委员会第一次全体会议、2019年12月6日二〇一九年苏州中学树莓社全体社员大会、2020年12月28日苏州中学树莓社第三届年度发展大会、2021年6月29日苏州中学树莓社社员大会、2022年1月4日苏州中学树莓社第四届年度全体社员大会、2023年1月7日苏州中学树莓社第五届社员大会、2024年1月10日苏州中学树莓社第六届社员大会和2024年苏州中学社树莓社第七届社员大会部分修正。
序 言
树莓谐音数媒，意为数字媒体。树莓社，一个关注于新时代数字媒体的社团，一个以影视创作为核心的艺术创作交流平台。 
树莓社的基本发展路线为：领导社员、团结全校、面向社会，以学习交流、创作实践为核心，坚持马克思主义文艺观，坚持“百花齐放、百家争鸣”基本方针，坚持民主集中制原则，维护专业影视制作业务的核心地位，探索数字媒体领域先进成果，发挥新闻传媒工作优势，推动树莓派长远目标建设，明确社团的社会责任，营造团结、紧张、严肃、活泼的社团氛围，自力更生、艰苦创业，立足苏中、面向大众，与时俱进、敢为人先，将树莓社建设成为一个关注于新时代数字媒体的社团、一个以影视创作为核心的艺术创作交流平台，为祖国的文艺事业培养更多未来的新兴力量。
以马雨璋、沈孙丰为核心的树莓社第一代领导集体，勇于创新、敢于斗争，使树莓社迅速壮大，管理体系不断完善，在社团文化繁荣的苏州中学占据了一席之地，为社团的稳定发展奠定基础。
二〇一八年九月，以马雨璋、张诗菡、沈孙丰为代表的树莓社筹备组成员经过长期准备，创建了苏州中学树莓社。树莓社的首任社长是马雨璋。在创社初期，树莓社的发展目标是壮大团队，定期以专栏形式拍摄短片、纪录片；组织团队参加国内外面向在读学生的大型影视创作比赛；在短期内获得TEDx@Suzhou校园大使的资格，并在近期围绕校园、城市、TED三大板块进行创作。
苏州中学树莓社坚持以影视行业的专业性作为社团发展的指导方向。微电影（影视创作）业务是苏州中学树莓社的根本业务。独立制作电影短片有助于浸润式深入地学习和享受影视创作过程。各项社团业务以微电影业务的发展为目标，各部门的设置均以微电影业务的需求为指导。苏州中学树莓社始终维护专业影视制作业务的核心地位。
二〇一八年十月，苏州中学树莓社开展了原创纪录片《苏迷》的拍摄活动，正式启动了树莓社原创纪录片的长期企划。树莓社希望以纪录片的形式展现苏州的真实生活和迷人风采，给苏州这座国家历史文化名城和著名风景旅游城市各方面的文化提供一片传播的沃土。希望更多的新苏州人通过此纪录片能够全面了解苏州，让更多生活在苏州的人看到未曾见过的苏州风采，让不熟悉苏州的人对苏州的风景、文化、魅力产生兴趣和喜爱。也希望树莓社的社员们通过参与此纪录片的制作，了解苏州、体验影视制作的基本流程。
十月中旬，树莓社应校方邀请，组织社员加入苏州中学运动会传媒中心，负责照片视频的拍摄，树莓社正式将校园新闻列入社团业务范围。校园新闻业务包括校内活动的摄影、相关短片的制作，以及在此基础上开设的校园电视台和《洱沐苏中》电视节目的制作和发布。校园新闻业务是苏州中学树莓社的核心业务，是树莓社生命力的基本保证。苏州中学运动会摄影志愿活动、新年音乐会拍摄活动是本项业务的代表。
经过实践，树莓社认识到，以学生社团的身份开展校园新闻拍摄等校内活动，带有很强的局限性。二〇一八年十一月，由陈天怡、马雨璋、沈孙丰、张诗菡等人组织筹备的苏州中学传媒中心正式成立。苏州中学传媒中心标识由张诗菡和周赟菲设计，借用校徽中松树的意象，采取更加青春活力的画风绘制，凸显苏中学生“活泼泼地”精神。苏州中学传媒中心是苏州中学树莓社的业务机构，负责分管校园电视台业务。为了充分发挥人才资源优势，苏州中学传媒中心由苏州中学树莓社和苏州中学Our Voice广播社共同运营，于二〇一八年十二月二十一日开播首期节目《洱沐苏中》。树莓希望用校园电视台的节目，与广播社的节目形成有机互补。以展现苏中学子风采，彰显苏中精神，丰富同学们的课余生活。节目旨在通过视频和自媒体传播，让同学、老师、家长和校友对苏中学生的日常、活动、精神风貌有更深入的了解，也通过镜头记录校园时事、展现出苏中风采人物的精神和各项校园活动的幕后故事。
二〇一九年七月，沈孙丰同志在树莓社管理层会议上正式提出建设全国性社会团体“树莓派”的长远发展目标。八月十五日，第一届树莓派组织委员会第一次全体会议正式召开，通过了《树莓派章程》（试行），并确立了在苏州地区建设分部、结交盟友，在全国范围内招收社员，五年内在苏州和周边城市创造足够影响力的阶段性目标。建设树莓派的长期计划正式启动。截至二〇一九年十二月，树莓派已建设有苏外树莓社分部、苏州六中树莓社分部，并招收到来自南京、上海、大庆、深圳、嘉兴乃至加拿大多伦多、澳大利亚悉尼等来自五湖四海的社员。树莓希望通过建设一个全国性的服务于高中生的数字媒体艺术交流社区，彰显数字媒体的生命力，传递影像的力量，培养中学生的兴趣爱好，给爱好者们提供学习交流的平台，为祖国的文艺传媒事业培养更多未来的新兴力量。
树莓派致力于打造一个全国性的属于高中生的数字媒体艺术交流平台。苏州中学树莓社致力于打造属于苏州中学学生的数字媒体艺术交流平台。总部设立在江苏省苏州中学，注册于苏州中学社团管理中心。
苏州中学树莓社的第二任社长是石允涵和刘恩岐。双社长制是树莓社历史上的一次伟大的尝试。以石允涵同志为主要代表的树莓社第二代领导集体，坚持社团的基本路线，实事求是，脚踏实地，明确了社团的社会责任，并对社团的运作模式进行了进一步的优化改革。正式推出“树莓社摄影志愿服务”这一活动品牌，得到校方和同学们的广泛认可。
苏州中学树莓社在第二届社员大会前完成了社团体制改革，实现了从兴趣小组式社团向学生创业社团的转变，明确了社团发展路线和社团定位，确定了社团组织形式和管理体制。组建苏州中学树莓社组织委员会作为社团的日常领导机构，提出建立在民主集中制原则上的社长岗位责任制和委员会制并行的社团架构。组委会以集体决议的形式记录社团重大事件，总结关键经验、促进团结进步；以社团年度总结和社长工作报告的方式，定期记录社团运作情况，为社团的持续发展保留更多有价值的资料。
在二〇一九年度的学校运动会中，苏州中学树莓社与学校在宣传领域开展了全面的合作。本次社团活动的参与人数达历史之最，超过25人；活动成果丰富多样，好评如潮。树莓社参与运动会现场转播，与专业团队一起完成开幕式的转播工作，刘昊知、沈睿韬等人成功实现航拍画面的实时转播；树莓社承包开幕式快剪短片制作任务，由马雨璋、沈孙丰、谭嘉怡三人合作的快剪视频，从拍摄到交付仅用时6小时，上线之后更是广受好评；树莓社承包本次运动会的全部图片摄影工作，创新地引入图片直播工作流程，将摄影与数字媒体再次结合起来，在保证高质量作品的同时，通过社团分工合作，进一步解决了时效难题，充分发挥团队、个人、科技三者各自优势，实现整体提升；树莓社拍摄运动会学生志愿者专题记录短片《英雄少年》，本片由编导部部长谭嘉怡担任编剧和导演，记录在运动会幕后默默奉献学生志愿者们，引发树莓社对志愿者身份的思考。本次活动是树莓社首次作为志愿者团队参与学校活动，也是树莓社社员首次以学校志愿者身份参与社团活动，这一次的经历对树莓社的社团活动思路造成了深远影响，推动了“树莓社摄影志愿服务”特色活动品牌的诞生。
照片直播业务由石允涵社长在学校运动会活动中首次提出。照片直播是依托互联网技术的一种新兴形式，借助这一技术，我们能够让拍摄的照片在五分钟之内送达同学、老师、家长手中。引进照片直播工作流程是树莓社新闻传播业务的重要发展成果。这标志着树莓社在社团核心业务下找到了适合学生社团的工作模式。照片直播业务得到了学校的大力支持，在2019校运会、校庆、环金鸡湖健走、奥体运动会等学校活动中，树莓社多次开设照片直播。目前已经形成了“摄影—审核—后期—上传”的标准工作流程，在每一个节点上专人负责，充分发挥社员优势。通过照片直播团队的组建，树莓找到了将个人优势转化为团队优势的办法，社员们体会到了分工合作的重要意义，培养了团队意识。
二〇二〇年一月，新冠疫情暴发。苏州中学树莓社组织募捐支援武汉、抗击疫情。在校外，以谭嘉怡为主的树莓派社员联合苏外、德威、园区校等学校社团发起大规模募捐活动；在校内，沈孙丰、刘恩岐联合商社社长张雨晨同学带领各社团发起募捐、开展义卖筹集资金。校内外合作开展，最终募集数十万元善款。树莓团队寻找合法合规的进货渠道，采购口罩、防护服等抗疫物资，并运抵武汉及周边城市。疫情得到控制后，树莓社拍摄了有关疫情的短片《夜班》和《看见》，由此来表达社员们对于此次新冠疫情的关心。
二〇二〇年上半年，树莓社多次开展影视制作活动。与校外组织合作，联合拍摄《霍尔顿·考尔菲德》《赎》等原创微电影短片。
二〇二〇年九月初，树莓社召开组委会会议。谭嘉怡、邢笑菡在会议上提出对社团部门设置进行优化。综合创社两年以来各部门活跃情况，根据社团设置部门的基本规则，将音乐部和配音部合并为表播部，更加契合社团发展的需要。
二〇二〇年九月，树莓社开展第三次社团纳新活动。策划与宣传部部长彭嘉仪同学对社团宣传方式进行了改革，大大提高了海报的宣传效率。彭嘉仪同学提出的要将宣传效益最大化的理念，打破了海报张贴的传统思路，对树莓社的宣传体系建设产生了深刻影响。
二〇二〇年十一月，TED×Youth@Suzhou城市青年大会在苏州中学来秀坊举办，由树莓社担任本次大会在校内的主要承办方。在完成照片直播等现有活动的基础上，树莓社额外负责了现场舞台灯光的设计工作。由杨乐天同学召集的，由张研南、江昊林等人组成的舞台灯光团队表现出众，广受好评。经过学校协调，树莓社舞台灯光业务正式形成，并成立业务机构灯光组，为学校各大活动提供舞台灯光服务。
二〇二〇年底，苏州中学树莓社被评为苏州市十佳社团、树莓社摄影志愿服务团队被评为苏州市阳光团队，石允涵被评为苏州市优秀社长。
苏州中学树莓社的第三任社长是朱奕。以叶卓、顾衡庭、朱奕同志为核心的树莓社第三代领导集体，坚持社团发展基本路线，在社团组织面临挑战之时，勇挑重任、敢于斗争，挽狂澜于既倒，扶大厦之将倾，带领社团走出发展困境，实现社团的全面发展。什么样的人适合领导树莓社、要如何领导树莓社、要建设什么样的树莓社，这些问题值得我们反复思考。树莓社第三届组委会经过不懈探索，通过划分核心业务和其他业务来明确社团发展方向，通过解释影视制作业务与校园新闻业务的关系指明了社团发展路线。提出“如何调动社员的积极性，是树莓社永不过时的课题”这一深刻论述。将社团文化和宣传建设从社团业务中独立出来，使得社员对社团文化和宣传建设产生更加强烈的热情。社团形象树莓酱、树莓b站账号、树莓公众号都在本年度得到了飞跃式的创新和发展。
二〇二一年一月，苏州中学传媒中心负责人李令剑发起《2021年苏州中学春节联欢晚会》节目摄制活动，并担任总导演。本次活动由苏州中学社团联盟主办，苏州中学传媒中心承办，苏州中学树莓社和Only One商社协办。树莓社主要负责节目编排、主持、宣传、摄制工作，由范奕凡、叶卓、朱奕、陆未一、杨乐天等人负责；商社负责财务和外联工作，主要由李多璨、王启华、王彦童等人负责。本次活动参与人数近百人，得到了学校和社会的大量关注，是树莓社和传媒中心举办大型晚会活动的一次大胆尝试。
二〇二一年三月，由李令剑和陈天一主持的苏州中学传媒中心广播节目《苏中曰》正式播出。本次节目通过精良的节目制作、优质的内容编排，得到了老师同学们的广泛好评。
二〇二一年四月二日，后期部顾衡庭同学开创了视频直播。这种新型传播形式，借助B站等互联网平台进行高画质的实时视频直播，让不在现场的观众与网友也可以身临其境。二〇二一年共进行三场视频直播，首次直播是和宅文化社合作的“【Lovelive】绽放吧，未来的光芒”二次元音乐会。经过多次的技术积累和改进，现已成为后期部视频直播标准化流程。
二〇二一年五月，由树莓派发展小组副组长李多璨同学和策划与宣传部部长陆未一同学联合发起的“破界”创意摄影作品征集活动正式举办。活动由来自苏中、星海、苏外、十中的数个社团共同举办，开辟了树莓社于校外社团广泛合作的先例，为树莓派的成立奠定基础。
二〇二一年五月十五日，树莓社召开扩大的第三届组委会第二次全体会议，研究讨论了当前时代背景下传媒中心（校园电视）的发展问题。树莓社认为，随着自媒体行业的快速发展，新闻传媒活动已经成为社团活跃度的基本保障，校园新闻节目的定义进一步扩大。树莓社的校园新闻业务可以跳出传统电视新闻的制作思路，大胆尝试创新。会议决定撤销苏州中学传媒中心在社团内的对应部门“新闻工作组”的建制，决定由组委会直接负责领导传媒中心运作，仅保留“苏州中学传媒中心负责人”一职。此次改革，是在社团内外环境发生变化的条件下，对树莓社校园新闻工作模式的重新部署，精简组织架构、提升运作效率、拓展创作思路，在时代变迁中更好地适应该项业务的发展需求。
二〇二一年暑假，由后期部顾衡庭同学策划、任星翰同学带领New Current编辑部撰稿、方芷昕同学配音的《苏迷》纪录片第一集《苏迷·清沁馈赠》重启了树莓社第一届管理层的纪录片企划。经过7月的前期策划与沟通，于8月10日在双塔定慧寺巷的一家绿豆汤店正式开拍，9月11日在B站树莓社官方账号上映，广受校内外好评。
二〇二一年九月三十日，苏州中学树莓社作为唯一的学生社团参与了苏州中学教育集团四校区联办的苏州奥体中心运动会的拍摄。
